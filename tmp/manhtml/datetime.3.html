NAME

FBB::User - Performs Date and Time Computations

SYNOPSIS

#include <bobcat/datetime>
Linking option: -lbobcat

DESCRIPTION

This class allows the programmer to manipulate date and time values. FBB::DateTime objects can be added to each other or subtracted from each other, converting their sums to UTC values. It is also possible to add or subtract seconds or tm structures to or from FBB::DateTime objects. This operation keeps the current time zone (UTC or local).

NAMESPACE

FBB
All constructors, members, operators and manipulators, mentioned in this man-page, are defined in the namespace FBB.

CONSTRUCTORS

  • DateTime(TimeType type = LOCALTIME):
    The default constructor, initialized to the current date and time. The default argument specifies that the conversion is performed to the machine's local time representation. Alternatively, UTC may be specified to represent the time as a Universal Time Coordinated time.
  • DateTime(time_t time, TimeType type = LOCALTIME):
    Initializes a FBB::DateTime object with information stored in the provided time_t value. The second argument specifies that the conversion is performed to the machine's local time representation. Alternatively, UTC may be specified to represent the time as a Universal Time Coordinated time.
  • DateTime(tm *tm, TimeType type = LOCALTIME):
    Initializes a FBB::DateTime object with information stored in the provided struct tm value. The second argument specifies that the conversion is performed to the machine's local time representation. Alternatively, UTC may be specified to represent the time as a Universal Time Coordinated time. The struct tm is defined as follows: 
    
    struct tm 
    {
        int tm_sec;     // seconds
        int tm_min;     // minutes
        int tm_hour;    // hours
        int tm_mday;    // day of the month
        int tm_mon;     // month
        int tm_year;    // year
        int tm_wday;    // day of the week
        int tm_yday;    // day in the year
        int tm_isdst;   // daylight saving time
    };
  • DateTime(DateTime const &other):
    The standard copy constructor is available by default.

OVERLOADED OPERATORS

  • operator bool() const:
    Returns true if the time decomposition could be performed without error. FBB::DateTime object use localtime_r(3) or gmtime_r(3) functions to break down the time_t values into elements. If the time could not be broken down, the error() member returns the error number (errno) associated with the error.
  • ostream &operator<<(ostream &str, FBB::DateTime) const &dt:
    Inserts a standard textual representation (without the trailing newline), of the time represented in the FBB::DateTime object into the indicated ostream. If the time value was broken down as LOCALTIME it is represented as local time, otherwise it's inserted as the Universal Time Coordinated time.
  • DateTime operator-(DateTime const &left, DateTime const &right):
    Subtracts two FBB::DateTime objects, and returns their difference in UTC format.
  • DateTime operator-(DateTime const &left, time_t seconds):
    Subtracts the number of seconds from the FBB::DateTime object, keeping the current time format (UTC or localtime).
  • DateTime operator-(DateTime const &left, tm const &fields):
    Subtracts the fields stored in the tm struct from the FBB::DateTime object, keeping the current time format (UTC or localtime).
  • DateTime &operator-=(DateTime const &right):
    Subtracts the argument's time values from the current FBB::DateTime object's time values, returning the difference in UTC format.
  • DateTime operator-=(time_t seconds):
    Subtracts the number of seconds from the FBB::DateTime object, keeping the current time format (UTC or localtime).
  • DateTime operator+(tm const &fields):
    Adds the fields stored in the tm struct to the FBB::DateTime object, keeping the current time format (UTC or localtime).
  • DateTime operator+(DateTime const &left, DateTime const &right):
    Adds two FBB::DateTime objects, and returns their difference in UTC format.

  • DateTime operator+(DateTime const &left, time_t seconds):
    Adds the number of seconds to the FBB::DateTime object, keeping the current time format (UTC or localtime).
  • DateTime operator+(DateTime const &left, tm const &fields):
    Adds the fields stored in the tm struct to the FBB::DateTime object, keeping the current time format (UTC or localtime).
  • DateTime &operator+=(DateTime const &right):
    Adds the argument's time values to the current FBB::DateTime object's time values, returning the difference in UTC format.
  • DateTime operator+=(time_t seconds):
    Adds the number of seconds to the FBB::DateTime object, keeping the current time format (UTC or localtime).
  • DateTime operator+(tm const &fields):
    Adds the fields stored in the tm struct to the FBB::DateTime object, keeping the current time format (UTC or localtime).

MEMBER FUNCTIONS

  • bool breakDown(TimeType type):
    Breaks up the time_t value stored in the FBB::DateTime object according to the specified time type. It returns true if the time could be broken down, false otherwise. The member error() returns the associated errno value.

    Except for following function error(), the members will not return meaningfull values if operator bool() returns false. All members returning a time-element do so according to the latest time-representation (i.e., LOCALTIME or UTC). All members returning numerical values use 0 as their lowest return values, except for the ...Nr() members, which start at 1.

  • unsigned error() const:
    Returns the errno value after the FBB::DateTime object construction. It can be interpreted by, e.g., FBB::Errno.
  • unsigned hours() const:
    Returns the number of hours of the time stored in a FBB::DateTime object (0-23).
  • unsigned minutes() const:
    Returns the number of minutes of the time stored in a FBB::DateTime object (0-59).
  • Month month() const:
    Returns the Month value of the time stored in a FBB::DateTime object.
  • unsigned monthDayNr() const:
    Returns the number of the day in the month of the time stored in a FBB::DateTime object (1-31).
  • unsigned seconds() const:
    Returns the number of seconds of the time stored in a FBB::DateTime object (0-59, but 60 and 61 may occur due to possible leap seconds).
  • time_t time() const:
    Returns the time_t values stored in the FBB::DateTime object itself.
  • WeekDay weekday() const:
    Returns the WeekDay value of the time stored in a FBB::DateTime object.
  • unsigned year() const:
    Returns the year element of the time stored in a FBB::DateTime object.
  • unsigned yearDay() const:
    Returns the day within the year of the time stored in a FBB::DateTime object. January 1 is returned as 0.
  • unsigned yearDayNr() const:
    Returns the day within the year of the time stored in a FBB::DateTime object. January 1 is returned as 1.
  • FBB:DayTime::TriVal dst() const:
    Returns an indication of an active Daylight Saving Time (DST) state for the (local) time represented in the FBB::DateTime object. When DST is active, the local time is one hour later as compared to the situation where DST is not active.
