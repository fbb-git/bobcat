NAME

FBB::TableSupport - Defines protocol for Table-support classes

SYNOPSIS

#include <bobcat/tableSupport>
Linking option: -lbobcat

DESCRIPTION

This class is designed as a (non-pure) base class for support-objects for FBB::Table objects. By overriding its virtual functions users have control over the way the actual layout of tables is defined. By default the virtual members of this class implement plain, blank-space delimited table elements.

When this is not appropriate, a class should be derived from FBB::TableSupport overriding those members that violate the programmer's current intentions. Then, an object of this derived class may be presented to one of the FBB::Table constructors, resulting in modifications of the default procedures used to disaplay separators between table elements.

NAMESPACE

FBB
All constructors, members, operators and manipulators, mentioned in this man-page, are defined in the namespace FBB.

NON_VIRTUAL MEMBER FUNCTIONS

  • void setParam(std::ostream &ostr, unsigned rows, std::vector<unsigned> const &colWidth, unsigned maxWidth):
    This member function defines the FBB::TableSupport's essential parameters:
    ostr is a reference to the std::ostream into which the table will be inserted;
    rows defines the number of rows used by the table;
    colWidth is a vector defining the widths of the individual columns of the table. Its values are ignored when the next parameter is unequal to 0, but in that case its size (its number of elements) are used;
    maxWidth is the width of each column. If set to 0 colWidth's elements are used to determine the width of the individual columns.

    Note that this member is normally not used by the programmer, but it is called with appropriate values by FBB::Table.

  • void setParam(unsigned maxWidth):
    This member function defines the FBB::TableSupport's maxWidth parameter, holding the width of each column.

    Note that this member is normally not used by the programmer, but it is called with appropriate values by FBB::Table.

ACCESSORS

  • unsigned tableWidth() const:
    Returns the width of the table.
  • unsigned nColumns() const:
    Returns the table's number of columns.
  • std::ostream &out() const:
    Returns the std::ostream into which the table is inserted.

VIRTUAL MEMBER FUNCTIONS

The following member functions may be overridden in derived classes. There default implementations are given together with their prototypes, below:

  • virtual void hline(unsigned row) const:
    This member is called just before the indicated row (offset, not number) is inserted into the std::ostream. Its line number is presented as a line-reference. Its intended task is to write a (partial) horizontal line either before the first row (row == 0) or between two rows of the table. By default it performs no actions.
  • virtual void hline() const:
    This member is called when the table's last line has been inserted into the std::ostream. Its intended task is to write a (partial) horizontal line at the end of the table. By default it performs no actions.
  • virtual void vline(unsigned col) const:
    This member is called just before the indicated column element (offset, not number) is inserted into the std::ostream. Its column number is presented as a reference. Its intended task is to write a separator either before the first element of a row (col == 0) or between two subsequent elements in the table's rows. By default it inserts one blank space.
  • virtual void vline() const:
    This member is called once the last element in a row has been inserted into the std::ostream. Its intended task is to write a line separator, terminating a line of the table. By default it inserts a new-line (\n) character.
