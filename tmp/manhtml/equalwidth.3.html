NAME

FBB::equalWidth - Manipulator class enforcing equal Table column widths

SYNOPSIS

#include <bobcat/equalwidth>
Linking option: -lbobcat

DESCRIPTION

An object of this class may be immediately inserted into an std::ostream to insert a FBB::Table whose columns have equal widths.

NAMESPACE

FBB
All constructors, members, operators and manipulators, mentioned in this man-page, are defined in the namespace FBB.

INHERITS FROM

-

CONSTRUCTORS

  • equalWidth(FBB::Table &table):
    The normal use of this constructor is within a insertion expression.
  • equalWidth(equalWidth const &other) throw():
    The standard copy constructor is available, but is normally not used.

MEMBER FUNCTIONS

There are no publicly available memberfunctions.

OVERLOADED OPERATORS

  • std::ostream &::operator<<(std::ostream &str, FBB::equalWidth const &ew):
    Tables may be inserted into std::ostream objects using this overloaded operator, which sets the column width to FBB::TableSpec::EqualWidth prior to inserting the table into the stream.

    Note that this operator is not defined in the namespace FBB.

The overloaded assignment operator is available, but is normally not used.
