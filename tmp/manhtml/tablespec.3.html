NAME

FBB::TableSpec - Base class for the class FBB::Table

SYNOPSIS

#include <bobcat/tablespec>
Linking option: -lbobcat

Note: #include <bobcat/tablespec> is implied by #include <bobcat/table>

DESCRIPTION

FBB::TableSpec is FBB::Table's base class. See table(3bobcat) for usage information. In this manual page only the specific characteristics of FBB::TableSpec are listed.

NAMESPACE

FBB
All constructors, members, operators and manipulators, mentioned in this man-page, are defined in the namespace FBB.

INHERITS FROM

-

ENUMERATIONS

The following enumerations are definedin th class FBB::TableSpec:

enum FillDirection
This enumeration holds two values:

  • Horizontal:
    Used to request row-wise storage of table elements in an FBB::Table.
  • Vertical:
    Used to request column-wise storage of table elements in an FBB::Table.

enum WidthType
This enumeration holds two values:

  • EqualWidth:
    Used to request equal width for all columns of an FBB::Table.
  • ColumnWidth:
    Used to request width-per-column for the columns of an FBB::Table.

CONSTRUCTORS

There are no public constructors in this class. FBB::TableSpec objects are constructed when FBB::Table objects are constructed.

OVERLOADED OPERATORS

  • std::ostream &operator<<(std::ostream &, FBB::TableSpec const &tableSpec):
    This overloaded operator is normally used to insert an FBB::Table into a std::ostream. See table(3bobcat) for details.

    Note that this operator is not defined in the namespace FBB.

The overloaded assignment operator is not available.

MEMBER FUNCTION

  • void setWidth(FBB::TableSpec::WidthType type):
    This member may be used to specify the width of indvidual or all columns of the tableSpec. Specify TableSpec::EqualWidth when all columns should have equal widths, specify TableSpec::ColumnWidth when each column should be as wide as its widest element.

MANPULATORS

  • FBB::equalWidth(FBB::TableSpec &tableSpec):
    This manipulator is normally used to insert an FBB::Table object into a std::ostream, enforcing equal column widths. To use this manipulator, the header file <bobcat/equalwidth> must be inserted.
  • FBB::columnWidth(FBB::TableSpec &tableSpec):
    This manipulator is normally used to insert an FBB::Table object into a std::ostream, enforcing column widths which are as wide as their widest elements. To use this manipulator, the header file <bobcat/columnwidth> must be inserted.
