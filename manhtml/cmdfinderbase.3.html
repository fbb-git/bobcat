NAME

FBB::CmdFinderBase - Base class for the class CmdFinder

SYNOPSIS

see the CmdFinder manpage

DESCRIPTION

This class is a base class for the class CmdFinder. All its accessible members and features are described in the CmdFinder manpage. This class supports move and copy construction and assignment operations.

NAMESPACE

FBB
All constructors, members, operators and manipulators, mentioned in this man-page, are defined in the namespace FBB.
